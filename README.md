# Spring Boot REST API - CRUD Application

## Description
This project is a RESTful API developed using Spring Boot. It performs basic CRUD (Create, Read, Update, Delete) operations with backend data storage.

The application includes input validation, security configuration, and global exception handling to ensure proper request processing and error management.

The API is connected to a MySQL database and tested using Postman.

## Features
- Create, Read, Update, and Delete operations
- Backend data storage using MySQL
- Request validation
- Basic security implementation
- Global exception handling
- RESTful API architecture

## Technologies Used
- Spring Boot
- MySQL
- Java
- Postman

## API Testing
All API endpoints were tested using Postman to verify request and response handling.

## Purpose
This project was developed to practice building REST APIs using Spring Boot, implementing CRUD operations, handling exceptions, and applying basic security with backend data storage.
